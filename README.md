
# URL rules for adjust documents on Mongodb:

```
    <!-- get all words -->
    (GET) http://localhost:3000 

    <!-- get word by ID -->
    (GET) http://localhost:3000/:id
    
    <!-- create word -->
    (POST) http://localhost:3000/

    <!-- update word by ID -->
    (PUT) http://localhost:3000/:id

    <!-- delete word by ID -->
    (DELETE) http://localhost:3000/:id
```
